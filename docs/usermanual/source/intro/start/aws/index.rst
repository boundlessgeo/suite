OpenGeo Suite for Amazon Web Services
=====================================

OpenGeo Suite is available for use with `Amazon Web Services <http://aws.amazon.com>`_ (AWS).

The standard method of purchasing and accessing OpenGeo Suite for AWS is through the **AWS Marketplace**. OpenGeo Suite is also available for GovCloud. If you are not sure which method is right for you, use the AWS Marketplace.
